OurHood

December 2019

OurHood is a private online platform where you can connect with your neighbours and community. Finally an easy way to share information including crime info, recommendations and tradesmen referrals, lost pets, local services and general news. OurHood is hosted on AWS using RDS for MariaDB, ECS storage and EC2 Instances, the Backend is Node JS with Express and the Frontend is Jquery, LESS, Handlebars and Bootstrap. The Mobile App was built using React Native and is a complete port of functionality from the website.
